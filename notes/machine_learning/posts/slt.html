Problem Setup

As usual, given some finite data \(D_n = \{x_1, x_2, \dots, x_n\}\) sampled approriately (i.i.d) from some unknown distribution \(q(x)\), we hope to find some parametrized model \(p(x | \theta)\) that approximates the true distribution \(q(x)\). In other words, our goal is to find some parameter \(\theta \in \Theta\) such that the KL divergence $$ \begin{align} KL(q(x) || p(x | \theta)) &= K(\theta) \\ &= \sum_x q(x)\log\frac{q(x)}{p(x|\theta)} \end{align} $$ is minimized (hopefully, vanishes).

For regular models \(p(x|\theta)\), such as linear regression, the KL divergence\(K(\theta)\) vanishes at some unique point \(\theta_0\) and thus the true parameter set \[\Theta_0 = \{\theta \in \Theta : K(\theta) = 0\}\] is precisely the point \(\theta_0\).
